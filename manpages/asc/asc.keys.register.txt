OVERVIEW: Registers App Store Connect API keys locally.

USAGE: asc keys register [--verbose ...] --key-id <key-id> --name <name> --path <path> --issuer-id <issuer-id>

OPTIONS:
  -v, --verbose           Activate verbose logging.
  -k, --key-id <key-id>   The key's id.
  -n, --name <name>       The key's name you. You can choose freely.
  -p, --path <path>       The absolute path to the p8 key file.
  -s, --issuer-id <issuer-id>
                          The id of the key issuer.
  --version               Show the version.
  -h, --help              Show help information.
